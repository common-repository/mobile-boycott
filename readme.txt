Widget for Syrian mobile companies boycott campaign.

== Description ==

Adds a nice image widget for Syrian Mobile Companies Boycott Campaign, for more information visit <a href="http://www.syrian-sport.net/nomobile/">NoMobile</a>.

== Installation ==

1. Upload the folder (mobile-boycott) to the plugins directory (wp-content/plugins).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Got to your widget page (www.example.com/wp-admin/widgets.php) and enable the widget.
